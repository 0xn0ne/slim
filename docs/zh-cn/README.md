
## 关于

> slim 是一个探索性质 python web 后端框架（底层基于aiohttp），它能够直接将 SQL 数据表直接绑定为 web 接口，并配有一个基于角色与数据表的管理权限模块。


### 特性

* 基于 aiohttp，全异步请求

* 映射数据表的增删改查为 web 接口

* 基于角色的权限管理，可控制数据表每一列的查询、读取、写入、新建和删除权限

* 命令行工具slim cli，可用于快速创建项目


### 限制

* Python 3.6+

* 目前仅支持 peewee 作为 ORM
