
## 关于

> slim 是一个基于 ASGI 的探索性质 python web 后端框架，它最大的特色是支持直接将 SQL 数据表直接绑定为 web 接口，并配有一个基于角色与数据表的管理权限模块。

> 同时，他也可以当做一个常规的高性能web框架来使用。

### 特性

* 全异步请求

* 支持函数和 class 两种组织 API 的形式

* 映射数据表的增删改查为 web 接口

* 基于角色的权限管理，可控制数据表每一列的查询、读取、写入、新建和删除权限

* 命令行工具slim cli，可用于快速创建项目


### 限制

* Python 3.6+

* 目前仅支持 peewee 作为 ORM
